html5_util_fctr.rb は create_db_tcp.rb で MySQL に登録された各 HTTP コネクション情報を
読み取り，HTML5 関連技術が利用されている HTTP コネクション数をカウントする
ためのものです．

やっていることは，html5 フィールドを読み，使われている HTML5 関連技術ごとに
カウンタを回し，CSV をはき出しているだけです．

使い方

% ruby html5_util_fctr.rb 対象DBテーブル名
出力結果は 対象DBテーブル名.csv となります．

その他，MySQL 接続先や DB 情報はソース中に記述してください．
